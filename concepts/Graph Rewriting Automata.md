# Graph Rewriting Automata

## Definition
An extension of cellular automata to **dynamic graph structures** where both node states AND topology evolve simultaneously via local rewriting rules. Unlike cellular automata with fixed lattices, GRA allow the network structure itself to change over time.

## Mathematical Formulation
GRA state: $G = \{\mathcal{A}, \mathcal{S}\}$
- $\mathcal{A}$: Adjacency matrix (topology)
- $\mathcal{S}$: State vector (node properties)

Both $\mathcal{A}$ and $\mathcal{S}$ evolve according to local rules depending on vertex state and neighborhood configuration.

## Key Extension from Cellular Automata
Standard [[Cellular Automata Framework]]:
- Fixed lattice topology
- State evolves, structure frozen

Graph Rewriting Automata:
- **Dynamic topology** - graph structure evolves
- **Topology-behavior coupling** - nodes' behavior constrained by structure; structure generated by behavior
- See [[Dynamic Graph Topology]] for detailed mechanism

## Fundamental Operations

### Vertex Division
Primary growth operation: vertex $v \to$ complete subgraph $K_n$
- Parent vertex divides into multiple daughter vertices
- Daughters inherit parent's connections
- Biological analog: [[Cell Division as Vertex Division]]
